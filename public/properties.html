Properties Requests

GET

  • 1

    /api/properties

This GET request will return a list of all available properties in the properties table.

Status 200

{"properties": [ { "property_id": id, "property_name": name, "location": location, "price_per_night": price, "host": host name}, ...] }

  • 2

    /api/properties/:id

This GET request will return a specific property, whose property_id matches the parametric endpoint.

Status 200

{"properties": [ { "property_id": id, "property_name": name, "location": location, "price_per_night": price, "host": host name}, ...] }
