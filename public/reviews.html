Reviews Requests

GET

  • 1

    /api/properties/:id/reviews

This GET request will return a list of all reviews posted for a specific property, based on the parametric endpoint.

Status 200

{"reviews": [{ "review_id": id,"comment": comment,"rating": rating,"created_at": date,"guest": guest name, "guest_avatar": img_url }, ... ], "average_rating": avg of ratings }

POST

  • 2

    /api/properties/:id/reviews

This POST request will post a new review for a specific property, based on the parametric endpoint.

Status 201

{ "review_id": id, "property_id": id, "guest_id": id,"rating": rating, "comment": comment, "created_at": date}
